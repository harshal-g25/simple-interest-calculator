# Simple Interest Calculator

This project is a simple interest calculator that calculates the interest earned on a principal amount over a period of time.

## Formula

Simple Interest = (Principal × Rate × Time) / 100

## Inputs
- Principal amount (P)
- Rate of interest (R)
- Time in years (T)

## Output
- Simple Interest calculated using the formula

## Example

If:
Principal = 1000  
Rate = 5%  
Time = 2 years  

Simple Interest = (1000 × 5 × 2) / 100 = 100
